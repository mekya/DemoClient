RTMP WebRTC Demo

Objectives:
In this demo, we publish media to Ant Media Server@ with RTMP and play media with WebRTC. Local media will play on left player and remote media will play on right player.

Directives:
  • Step 1: Be sure AntMediaServer run @. If it is wrong change it from here.
  • Step 2: Select a media file to broadcast. Then publishing will start automatically. Remote player will play when server become ready.
  • Step 3: You can play streamed media with another device or new browser page by clicking or reading generated QR Code.
  • Step 4: You can stop broadcasting by clicking stop.
